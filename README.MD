## Install
```
composer install
npm install
npm run dev
php artisan migrate
php artisan db:seed
php artisan db:seed --class=MatchingSeeder
php artisan passport:install
php artisan config:cache
```

## update
```
composer update
npm update
npm run dev
php artisan migrate
php artisan db:seed
php artisan db:seed --class=MatchingSeeder
php artisan config:cache
```
